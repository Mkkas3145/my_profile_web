저는 웹 개발자로서 다양한 기술 스택을 활용하여 여러 프로젝트를 성공적으로 수행한 경험이 있습니다. 특히, 2년 동안 수익 창출이 가능한 플랫폼인 Louibooks를 직접 운영하고 개발한 경험이 있습니다. 이 경험을 통해 복잡한 웹 애플리케이션을 설계하고 구현하는 능력을 키웠습니다.
개인 프로젝트
Louibooks
기존 양산형 자유 연재 플랫폼의 한계를 극복하고 강력한 현지화 기능을 탑재하여 사용자들이 소설, 만화, 웹툰, 애니메이션 콘텐츠를 자유롭게 연재하고 소비하고 수익 창출할 수 있게 하는 거대한 프로젝트입니다.


기능들을 간략하게 소개하면 다음과 같습니다.
구글 애드센스 연동(광고), 토스 연동, 클라우드 기능(폴더, 텍스트, 이미지 저장)
소설 뷰어(TTS 기능 포함), 만화 뷰어, 웹툰 뷰어, 애니메이션 뷰어
작품 내 커뮤니티 기능(유튜브와 비슷), 작품 목록 기능, 작품 목록 정렬 기능.
작품 만들기, 작품 수정, 작품 현지화 기능, 계정 관리 기능, 이미지 업스케일링 기능
접근 제한(허용된 장소 또는 지에서만 접근 가능), 2차 인증, 활성화된 세션 관리하기
언어 변경, 사용자 번역본 제출 기능, 최근 기록, 검색 기록, 알림(답글, 업로드, 공지), 구독 기능.
작품 탐색, 작품 분석(전체 조회수, 조회수 증가, 연속 보기율, 노출 수, 클릭률, 평점, 예상 수익), 작품 소비 남녀 비율, 작품 소비 나이대별 비율.
사용자 맞춤 콘텐츠 기능(캐싱 방식).

그 외 소개되지 않은 많은 기능들이 있을 수 있습니다, 복구하면서 작동하지 않는 기능이 의외로 많습니다.

Web Video Player
이 프로젝트는 사용자 경험(UX)을 극대화한 비디오 플레이어를 개발하는 것을 목표로 합니다. 현대적인 웹 환경에서 동영상 콘텐츠는 중요한 요소이며, 이에 따라 사용자들이 쉽게 접근하고 즐길 수 있는 직관적이고 반응성이 높은 비디오 플레이어의 필요성이 증가하고 있습니다. 본 프로젝트는 다양한 기기와 브라우저에서 원활하게 작동하며, 사용자의 편의를 최우선으로 고려한 기능을 갖춘 비디오 플레이어를 개발하고 배포하는 것을 목표로 합니다.


기능들을 간략하게 소개하면 다음과 같습니다.
재생(HLS 지원) 및 일시정지, 음소거 및 볼륨 조절, 전체화면, PIP 및 화면 확장
동적 조명 및 레터박스 채우기
미리보기, 단축키, 컨텍스트 메뉴

기능들이 계속해서 개발되는 중이므로 자세한 기능 설명은 추후에 추가될 예정입니다.

습득한 언어
Java
자바는 썬 마이크로시스템즈의 제임스 고슬링(James Gosling)과 다른 연구원들이 개발한 객체 지향적 프로그래밍 언어이다.
Kotlin
코틀린은 젯브레인즈사에서 개발하는 크로스 플랫폼 범용 프로그래밍 언어로, 자바와 완벽하게 호환되게 설계되었으며 일반적으로 JVM에서 사용되나 자바스크립트, LLVM 그리고 데이터 사이언스 영역에서도 사용 가능하다.
JavaScript
자바스크립트는 객체 기반의 스크립트 프로그래밍 언어이다. 이 언어는 웹 브라우저 내에서 주로 사용되며, 다른 응용 프로그램의 내장 객체에도 접근할 수 있는 기능을 가지고 있다.
TypeScript
타입스크립트는 자바스크립트의 슈퍼셋인 오픈소스 프로그래밍 언어이다. 마이크로소프트에서 개발, 유지하고 있으며 엄격한 문법을 지원한다. C#의 리드 아키텍트이자 델파이, 터보 파스칼의 창시자인 Anders Hejlsberg이(가) 개발에 참여한다.
HTML
하이퍼 텍스트 마크업 언어는 웹 페이지 표시를 위해 개발된 지배적인 마크업 언어다. 또한, HTML은 제목, 단락, 목록 등과 같은 본문을 위한 구조적 의미를 나타내는 것뿐만 아니라 링크, 인용과 그 밖의 항목으로 구조적 문서를 만들 수 있는 방법을 제공한다.
CSS
종속형 시트 또는 캐스케이딩 스타일 시트는 마크업 언어가 실제 표시되는 방법을 기술하는 스타일 언어로, HTML과 XHTML에 주로 쓰이며, XML에서도 사용할 수 있다.
Web Components
웹 컴포넌트는 개개의 HTML 요소들의 캡슐화와 상호운용을 가능케 하는 웹을 위한 표준 컴포넌트 모델을 제공하는 기능들의 모임이다.
Python
파이썬은 1991년 네덜란드계 소프트웨어 엔지니어인 귀도 반 로섬이 발표한 고급 프로그래밍 언어로, '인터프리터를 사용하는 객체지향 언어'이자 플랫폼에 독립적인, 동적 타이핑(dynamically typed) 대화형 언어이다.
Golang
Go는 2009년 구글에서 일하는 로버트 그리즈머, 롭 파이크, 켄 톰프슨이 개발한 프로그래밍 언어이다. 가비지 컬렉션 기능이 있고, 병행성(concurrent)을 잘 지원하는 컴파일 언어다. 구문이 C와 비슷하지만 메모리 보안, 쓰레기 수집, 구조 타이핑, CSP 스타일 병행성을 제공한다.
SQL
SQL은(는) 관계형 데이터베이스 관리 시스템(RDBMS)의 데이터를 관리하기 위해 설계한 특수 목적의 프로그래밍 언어이다.
